# SeedX

SeedX is an AI-assisted autonomous drone system designed for large-scale reforestation. The project focuses on identifying deforested or degraded land and autonomously dispersing seeds to support ecosystem restoration and sustainable afforestation.

## The Problem

Deforestation continues to damage ecosystems, reduce biodiversity, and impact environmental sustainability. Traditional reforestation methods can often be slow, labor-intensive, and difficult to scale.

## Our Solution

SeedX combines drone technology, AI, and environmental monitoring systems to create a smarter approach to reforestation.

The system is designed to:

* Detect deforested or degraded regions
* Disperse seeds using an automated seed-dropping mechanism
* Monitor planted regions using NDVI technology

## Key Features

* Autonomous drone-based navigation
* AI-assisted deforestation detection
* Precision seed dispersion system
* NDVI-based plant growth monitoring

## AI Workflow

1. Collection of aerial images of forested and deforested regions
2. Image annotation using Roboflow
3. Dataset export in YOLOv8 format
4. Model training using Ultralytics YOLOv8
5. Evaluation using precision and recall metrics

## Challenges Faced

* Drone stability and limited flight duration
* Sensor accuracy and calibration
* Improving AI model precision
* Designing an efficient seed dispersion mechanism

## Potential Impact

* Large-scale afforestation
* Faster and smarter reforestation methods
* Long-term ecological restoration
* Sustainable environmental solutions
